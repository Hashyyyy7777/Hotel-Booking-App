# Hotel Booking App

A modern and responsive hotel booking application that allows users to browse hotels, view available rooms, and manage reservations seamlessly. The project is built using modern web technologies to provide a smooth and user-friendly booking experience.

## 🚀 Features


* View hotel details and room information
* Responsive design for desktop and mobile devices
* User-friendly booking interface
* Booking history management
* Modern and clean UI

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript
* React.js
